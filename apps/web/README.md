# NodeAdmin Web 🌐 [EN/ZH]

Premium React-based management dashboard for NodeAdmin Pro. Optimized for performance and sleek user experience.
NodeAdmin Pro 的高级 React 管理面板。致力于提供极致性能与流畅的用户体验。

---

## 🚀 Features | 功能

- **Rich Aesthetics**: Vibrant, dark-mode first UI with modern color palettes.
  **精美视觉效果**：生动的、暗黑优先的现代调色板与 UI。
- **Recharts Integration**: Beautiful, real-time data visualizations of node health and usage.
  **Recharts 集成**：精美的图表实时展现节点健康与流量分配情况。
- **Bilingual Support**: Fully localized in English and Simplified Chinese (i18next).
  **双语支持**：完整支持中英文切换。
- **SDK-Powered**: Direct integration with `@airport/sdk` for clean API interactions.
  **SDK 驱动**：通过直接集成 `@airport/sdk` 实现简洁、类型安全的 API 通讯。

---

## 🛠️ Setup | 设置

### Prerequisites | 前置要求
- Node.js 18+
- Vite Knowledge

### Quick Start | 快速启动
```bash
# Install dependencies | 安装依赖
npm install

# Start Dev Server | 启动开发服务器
npm run dev
```

---

## 📁 Structure | 项目结构

- **components**: Atomic UI components and feature-specific widgets.
  **components**: 原子化的 UI 组件与特定功能的小部件。
- **views**: Main page layouts (Dashboard, Nodes, Users, etc.).
  **views**: 页面主布局（仪表盘、节点、用户等）。
- **api**: API client helper and standardized request builders.
  **api**: API 客户端辅助工具与标准的请求构造器。
